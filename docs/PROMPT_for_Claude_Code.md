## ДИЗАЙН-СИСТЕМА (строго соблюдать)

### Палитра — глубокий «deep tech» дарк
```
--bg:        #0A0A0F   /* основной фон, почти чёрный с холодным подтоном */
--surface:   #14141C   /* карточки, секции */
--surface-2: #1C1C26   /* приподнятые элементы, hover */
--border:    #2A2A38   /* тонкие разделители, 1px */
--text:      #F4F4F7   /* основной текст */
--muted:     #8A8A99   /* вторичный текст, подписи */
--accent-1:  #5B8DEF   /* электрик-блю */
--accent-2:  #A78BFA   /* фиолет */
--accent-3:  #2DD4BF   /* тил — точечные акценты, цифры */
```
- Фирменный градиент: `linear-gradient(120deg, #5B8DEF 0%, #A78BFA 100%)` — для ключевых слов в заголовках, кнопок, линий.
- Фон не плоский: мягкое radial-свечение акцентным цветом за hero и за блоком контактов (большое, размытое, ~6% непрозрачности).
- Точечный тил `#2DD4BF` только на цифрах-результатах и иконках — не злоупотреблять.

### Типографика (важно: полная поддержка кириллицы)
- **Дисплейные заголовки:** `Unbounded` (Google Fonts, есть кириллица) — крупно, плотно, uppercase для H1/секций.
- **Текст:** `Manrope` (Google Fonts, кириллица) — для body, подписей, кнопок.
- Масштаб: H1 — clamp(2.6rem, 7vw, 6rem); H2 — clamp(2rem, 4.5vw, 3.5rem); body — 1.0625rem; line-height 1.5; межбуквенный интервал заголовков -0.02em.
- Контраст веса: заголовки 700–800, тело 400–500.

### Сетка и ритм
- Контейнер max-width 1200px, поля 24px (мобайл) / 48px (десктоп).
- Между секциями: 14–20vh «воздуха».
- Скругления 16–20px на карточках, 9999px на кнопках/чипах.
- Тонкие 1px бордеры `--border` вместо теней; тени только мягкие и большие.

### Видео-фоны секций (для 2/3/4)
- Autoplay, muted, loop, playsInline; `object-cover`, заполняют секцию.
- Поверх — тёмный градиент-оверлей (`--bg` с альфой 0.55–0.75), чтобы текст уверенно читался (контраст AA).
- Лёгкий `backdrop` / виньетка по краям. Видео — атмосфера, не главный герой: текст и метрики важнее.

### Микро-детали
- Ховер на интерактиве — лёгкое поднятие (translateY -2px) + бордер в акцент, transition 200ms ease.
- Прогресс-полоса скролла сверху, заполняется фирменным градиентом.

## АНИМАЦИИ «ВЫЕЗД С КРАЁВ» (ядро ощущения)

Переиспользуемый компонент `components/Reveal.tsx` на Framer Motion:

- Проп `from`: `"left" | "right" | "bottom" | "top"`.
- `whileInView`: элемент выезжает из-за края — начальное `opacity: 0` + смещение (`x: -80` для left, `x: 80` для right, `y: 60` для bottom).
- Пружина мягкая: `transition: { type: "spring", stiffness: 90, damping: 20, mass: 0.8 }` либо `ease: [0.22, 1, 0.36, 1]`, duration ~0.8s.
- `viewport={{ once: true, margin: "-15% 0px" }}`.
- Проп `delay` для каскада (заголовок → подзаголовок → карточки, шаг 0.08–0.12s).

**Применение по секциям:**
- Заголовки секций — **слева** (`from="left"`).
- Описания/лиды — **справа** (`from="right"`).
- Карточки и метрики — **снизу** (`from="bottom"`), каскадом со stagger.
- В hero — текст всплывает снизу с blur-to-clear (`filter: blur(8px) → blur(0)`).

Группы карточек оборачивай в stagger-контейнер (`staggerChildren: 0.1`). Уважай `prefers-reduced-motion` — при флаге только простой fade.

## SCROLL-HERO (видео-скраб) — точные требования

1. Видео уже на месте: `/public/hero.mp4` (бывш. `1.mp4`).
2. Создай компонент `app/components/ScrollHero.tsx` (или `src/components/ScrollHero.tsx`, если используешь `src/`).
3. Компонент должен:
   - Рендерить высокий контейнер высотой **300vh** — дистанция скролла для прокрутки видео.
   - Внутри — **sticky**-видеоэлемент, заполняющий вьюпорт (`100vh`, object-cover).
   - Видео без звука, по умолчанию на паузе, источник `/hero.mp4`.
   - На монтировании — scroll-listener, считающий прогресс прокрутки внутри контейнера от **0 до 1**.
   - Маппить прогресс в `video.currentTime` (`progress * video.duration`).
   - `requestAnimationFrame`-цикл с **lerp-сглаживанием, фактор 0.22** — плейхед скользит, а не дёргается по кадрам.
   - Очищать scroll-listener и animation frame на размонтировании.
4. Атрибуты видео: `playsInline`, `muted`, `preload="auto"`.
5. Fade-in первые **200ms** после монтирования.
6. ScrollHero — **первая секция** на главной (`app/page.tsx`).

**Поверх hero (sticky-слой):**
- Крупный H1 «Алексей Фролов» и сабхедлайн, всплывающие снизу с blur-to-clear.
- Внизу — мягко пульсирующий индикатор «Листайте ↓».
- Тёмный градиент-оверлей снизу видео для читаемости текста.
